# Projeto pessoal simples

Este projeto tem o intuito de me facilitar em alguns processos de preenchimento de alguns formularios repetitivos.

> ## Método de funcionamento

Será necessário carregar um ficheiro `.csv` ou um `.xlsx` com um template pré definido. Cada coluna do ficheiro será um campo novo a ser marcado preenchido, por isso é importante que todos os dados estejam bem estruturados.
